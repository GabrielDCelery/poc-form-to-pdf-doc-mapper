3. CONFIGURATION BASICS
The configuration tells the form builder what to do. It is a set of rules that gets interpreted
by the form builder to set up the structure, the styling and the form’s internal logic.
It also gets inserted into
the “Embedded assets – Javascript” section, underneath the
code that you added in the previous step.
